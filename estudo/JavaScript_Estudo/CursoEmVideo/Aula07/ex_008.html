Operadores de Atribuíção:

  1. O operador simples:
  2. = (igual no JavaScript é traduzido para recebe), por exemplo:

  3. var x = 5+4**3 (traduzindo a variavel x recebe o valor de 69)
  4. ou, var x = (5+4)**3 (traduzindo a variavel x recebe o valor de 729)

  5. Operador de Auto-atribuição:

  6. Podemos fazer dois tipos de auto atribuição, vale ressaltal os tipos delas:

    1. A por extenso:

    2. var n = 3
      n=n+4 (ou seja agora o valor de n agora é 7)

    3. E podemos também chegar no mesmo resultado de forma resumida:

      var n =3
      n+=4 (n também será 7)

  1. Atribuição de Incremento (operdor de incremento):

  2. Existem dois tipos de operadores de atribuição por incremento:


    1. Fazer por metodologia de Auto-atribuição. Exemplo:
      var y = 5
      y+=1 (y fica com o valor de 6) adicionou 1
      y-=1 (y fica igual a 5 novamente)subtraiu 1


    2. Ou podemos demonstrar ambos os calculos com um operador de pós-incremento. Exemplo:
      var y = 5
      y++ (y fica igual a 6 ) adicionou 1
      y-- (y fica igual a 5 novamente) subtraiu 1
