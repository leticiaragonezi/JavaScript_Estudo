Operadores Relacionais:

  1. Simbologia de operadores relacionais:
  2. > (Maior que )
  3. < (Menor que)
  4. >= (Maior ou igual que)
  5. <= (Menor ou igual que)
  6. == (Igual)
  7. != (Diferente)

  8. OBS: A saída de todas os cálculos realizados com tais operadores retornarão valores booleanos:
    (True ou False).

    1. Operador de identidade ou Operador de Igualdade Restrita:
    2. === (saber se o código é identico(typeof))
    3. !== (desigual restrito compara se eles são de tipos diferentes(typeof) )
    4. OBS: o teste além de comparar se os valores são iguais, ele comparára
      se o tipo da variavel também será a mesma.
